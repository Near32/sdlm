## Overview

Prompt-inversion benchmark suite: optimizes a soft-prompt (differentiable token distribution) to
reconstruct a target sentence. Supports STGS, REINFORCE, SODA, GCG, and O2P gradient estimators.
Entry points are `invert_sentences.py` (single target) and `batch_optimize_main.py` (batched
experiments). Core optimization logic lives in `main.py`; baseline implementations in `baselines/`.

### `lora_logits`

LoRA-factored prompt logits: replaces the flat `(1, L, V)` free_logits tensor with a rank-r factorisation `lora_A @ lora_B` where `lora_A` is `(1, L, r)` and `lora_B` is `(1, r, V)`. Reduces parameters from L·V to r·(L+V) and couples gradient signals across positions through shared B rows. Compatible with `logit_decay` and `discrete_reinit`. **Locations**: `main.py`, `invert_sentences.py`, `batch_optimize_main.py`.

| `--logits_lora_b_learning_rate` | Optional separate LR for LoRA matrix B (default: None = same as `--learning_rate`) |
